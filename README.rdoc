== README

== ОПИСАНИЕ

Web-приложение "Around World" показывает список стран (и городов, входящих в них), которые посетил пользователь.
Каждый пользователь может менять только свои списки.

В будущем будут добавлены функции:
* сравнения посещенных стран и городов между пользователя.
* отображение посещенных мест на картах Google.

== ДЛЯ РАЗРАБОТЧИКОВ

В качестве базы данных используется MongoDB.

Приложение состоит из трех моделей:
User
Country
City
